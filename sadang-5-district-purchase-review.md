## 6. 투자성 검토

### 6.1 11억원 매수가의 의미

이 매물은 일반적인 구축 다세대 실거주 매수라기보다 **재건축 조합원 권리 취득 목적의 투자**로 보아야 합니다.

따라서 11억원이 비싼지 싼지는 다음 공식에 가깝게 판단해야 합니다.

```text
총투입금 = 매매가 11억원 + 취득세/중개수수료/기타비용 + 추가분담금 - 회수 가능한 임대수익
투자성과 = 신축 배정 주택의 예상 가치 - 총투입금 - 시간/리스크 비용
```

현재는 추가분담금과 배정 평형이 확인되지 않았으므로 총투입금이 확정되지 않습니다.

정비몽땅 사당5 사업장에는 추정분담금 조회 페이지가 공개되어 있고, 조합원 분담금 산식은 다음 구조로 안내되어 있습니다.

```text
조합원분담금 = 조합원분양가 - 종전자산권리가액
종전자산권리가액 = 조합원 종전자산 감정평가액 * 비례율
비례율 = (분양수입총액 - 사업비총액) / 종전자산감정평가액 총액
```

출처: [정비몽땅 사당5 추정분담금 정보](https://cleanup.seoul.go.kr/cafe/mastr-cleanup-estmBndn/execute.do?cafeId=590900000601b27)

따라서 이 매물의 투자성은 단순 매매가보다 **종전자산 감정평가액, 비례율, 조합원분양가**를 확인한 뒤 판단해야 합니다.

### 6.3 84타입 조합원분양 시나리오

정비몽땅 평면도 메뉴에는 84A, 84B, 84C 타입이 공개되어 있습니다. 다만 이는 사업장에 공개된 주택형 목록일 뿐이며, 이 매물의 조합원이 84타입을 신청하거나 배정받을 수 있다는 뜻은 아닙니다.

84타입 조합원분양을 전제로 검토하려면 아래 네 가지가 먼저 확인되어야 합니다.

| 확인 항목 | 의미 | 확인 전 판단 |
| --- | --- | --- |
| 84타입 신청 가능 여부 | 이 물건의 권리로 84A/B/C 신청이 가능한지 | 미확인 |
| 84타입 조합원분양가 | 조합원이 부담하는 84타입 공급가격 | 미확인 |
| 종전자산 감정평가액 | 이 물건의 조합 내 평가액 | 미확인 |
| 비례율 | 권리가액 산정에 적용되는 비율 | 미확인 |

정비몽땅 추정분담금 산식을 적용하면 84타입 기준 추가분담금은 다음처럼 계산됩니다.

```text
84타입 추가분담금 = 84타입 조합원분양가 - (종전자산 감정평가액 * 비례율)
84타입 총투입금 = 매매가 11억원 + 취득/중개/보유비용 + 84타입 추가분담금
84타입 투자여력 = 예상 신축가치 - 84타입 총투입금
```

실무적으로는 아래 표를 채우면 됩니다.

| 항목 | 값 | 비고 |
| --- | --- | --- |
| 매매가 | 11억원 | 사용자 제공 매물가 |
| 취득/중개/기타비용 | 확인 필요 | 취득세, 중개보수, 법무비 등 |
| 84타입 조합원분양가 | 확인 필요 | 조합 분양신청 안내문 또는 추정분담금 자료 |
| 종전자산 감정평가액 | 확인 필요 | 감정평가 통지 또는 조합 확인 |
| 비례율 | 확인 필요 | 관리처분계획 또는 추정분담금 자료 |
| 84타입 추가분담금 | 계산 필요 | 조합원분양가 - 권리가액 |
| 84타입 총투입금 | 계산 필요 | 매매가 + 비용 + 추가분담금 |
| 준공 후 예상가치 | 별도 조사 필요 | 주변 신축/준신축 84타입 실거래 비교 |

판단 기준은 단순합니다.

- **긍정적 시나리오:** 84타입 신청 가능, 권리가액이 충분해 추가분담금이 감당 가능한 수준, 총투입금 대비 준공 후 예상가치에 여유가 있는 경우
- **중립 시나리오:** 84타입 신청은 가능하지만 추가분담금이 커서 총투입금이 주변 신축가치에 근접하는 경우
- **부정적 시나리오:** 84타입 신청이 불가능하거나, 가능하더라도 총투입금이 준공 후 예상가치와 비슷하거나 더 높아지는 경우

따라서 84타입 시나리오의 핵심 질문은 “84타입을 받을 수 있느냐”보다 한 단계 더 나아가 **“84타입을 받았을 때 총투입금이 얼마가 되느냐”**입니다. 현재 문서 기준으로는 84타입 자체는 공식 평면도에 존재하지만, 이 매물의 84타입 배정 가능성과 추가분담금은 아직 확인되지 않았습니다.

### 6.4 현재 판단

현재 정보만 기준으로는 **조건부 검토 가능**입니다.

다만 아래 조건 중 하나라도 불리하면 11억원 매수가는 공격적일 수 있습니다.

- 조합원 지위 승계가 불가능한 경우
- 59㎡ 이하만 신청 가능한 경우
- 84타입 신청은 가능하지만 추가분담금이 과도한 경우
- 종전자산 감정평가액이 낮아 추가분담금이 크게 발생하는 경우
- 관리처분인가, 이주, 착공 일정이 장기 지연되는 경우

## 7. 핵심 리스크

### 7.1 조합원 지위 승계 리스크

서울 재건축 매수에서 가장 먼저 확인해야 할 리스크입니다.

도시 및 주거환경정비법은 조합원 자격, 조합원 지위 양도, 투기과열지구 내 거래 제한과 관련된 규정을 두고 있습니다. 특히 조합설립인가 이후 재건축 물건을 매수할 때는 조합원 지위 승계 가능 여부를 법무사나 변호사, 조합을 통해 확인해야 합니다.

확인할 자료:

- 조합원 명부상 매도인 등재 여부
- 매도인의 보유기간, 거주기간, 1세대 1주택 여부
- 조합원 지위 양도 예외사유 해당 여부
- 조합의 조합원 지위 승계 확인서 또는 이에 준하는 공식 회신

관련 법령: [도시 및 주거환경정비법 제39조](https://www.law.go.kr/법령/도시%20및%20주거환경정비법/제39조)

### 7.2 현금청산 리스크

조합원 지위를 승계하지 못하거나 분양신청 절차에서 요건을 충족하지 못하면 신축 주택 배정이 아니라 현금청산 문제가 생길 수 있습니다.

관련 법령:

- [도시 및 주거환경정비법 제72조](https://www.law.go.kr/법령/도시%20및%20주거환경정비법/제72조)
- [도시 및 주거환경정비법 제73조](https://www.law.go.kr/법령/도시%20및%20주거환경정비법/제73조)

### 7.3 추가분담금 리스크

11억원에 매수하더라도 신축 배정 시 추가분담금이 발생할 수 있습니다. 추가분담금은 종전자산 감정평가액, 조합원 분양가, 사업비, 일반분양 수입 등에 따라 달라집니다.

확인 전까지는 투자수익률을 확정할 수 없습니다.

### 7.4 사업 지연 리스크

정비몽땅 사업장검색 기준 진행단계는 사업시행인가이고, 추진경과 기준으로는 2024년 10월 17일 사업시행계획변경인가까지 확인됩니다. 사업시행인가 이후에도 일반적으로 관리처분인가, 이주, 철거, 착공, 준공 절차가 남습니다.

따라서 단기 시세차익보다는 장기 자금계획이 필요합니다.

### 7.5 대출·세금 리스크

재건축 구역 내 주택 매수는 대출 규제, 취득세, 보유세, 양도세, 입주권 관련 세무 판단이 복합적으로 얽힐 수 있습니다. 특히 투자 목적이면 개인의 주택 수와 기존 보유 부동산에 따라 세후 수익률이 달라집니다.

## 8. 계약 전 확인 체크리스트

### 8.1 조합 또는 정비몽땅에서 확인

- [ ] 이 물건의 매도인이 조합원 명부에 등재되어 있는가
- [ ] 매수인에게 조합원 지위가 승계되는가
- [ ] 승계 제한이나 예외사유 검토가 끝났는가
- [ ] 종전자산 감정평가액은 얼마인가
- [ ] 신청 가능한 평형은 무엇인가
- [ ] 조합원 분양가 또는 추정분담금 자료가 있는가
- [ ] 관리처분계획 수립 일정은 어떻게 되는가

### 8.2 매도인에게 받을 자료

- [ ] 등기부등본
- [ ] 건축물대장
- [ ] 토지대장 또는 대지권 관련 자료
- [ ] 조합원 지위 관련 조합 확인 자료
- [ ] 종전자산 감정평가 통지 또는 관련 안내문
- [ ] 분양신청 안내문
- [ ] 체납 관리비, 조합비, 대여금 부담 여부

### 8.3 계약서 특약 예시

아래는 검토용 문구입니다. 실제 계약에는 공인중개사, 법무사, 변호사 검토가 필요합니다.

```text
본 계약은 매수인의 조합원 지위 승계가 적법하게 가능한 것을 전제로 한다.

```
